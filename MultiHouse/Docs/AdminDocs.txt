
Admin Guarded Paths(must be, if not, say to developer):
    /House/Create
    /House/HouseUpload
    /Request
    /Request/RequestView/{id}
    /Request/EditStatus/{id}
    
    
    
Приступая к работе:
 
 1) заходим на сайт, прописываем в url путь /admin
 2) вводим пароль => используем админскую панель
 
Настройка сайта:

    - Настройка пароля:
        переходим по пути Admin/PasswordSetView
        вводим старый пароль (по умолчанию 123, обязательна смена пароля),
        придумываем новый( обязательно запоминаем его).
        
    - Настройка уведомлений с почты(обязательно, иначе отправка 
                             сообщений пользователю не будет работать)
        вводим пароль админа сайта,
        вводим логин почты, 
        вводим пароль почты.
        
        + нужно удоствериться что сторонним приложениям дан доступ 
          к использованию почты
