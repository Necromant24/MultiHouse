
Admin Guarded Paths(must be, if not, say to developer):
    /House/Create
    /House/HouseUpload
    /Request
    /Request/RequestView/{id}
    /Request/EditStatus/{id}
    /House/DeleteView
    
    
    
Приступая к работе:
 
 1) заходим на сайт, прописываем в url путь /admin
 2) вводим пароль => используем админскую панель
 
Настройка сайта:

    - Настройка пароля:
        переходим по пути Admin/PasswordSetView
        вводим старый пароль (по умолчанию 123, обязательна смена пароля),
        придумываем новый( обязательно запоминаем его).
        
    - Настройка уведомлений с почты(обязательно, иначе отправка 
                             сообщений пользователю не будет работать 
                             и будет вылетать ошибка у клиента)
                             
        переходим на url /Admin/EmailSetView
        вводим пароль админа сайта,
        вводим логин почты, 
        вводим пароль почты.
        
        + нужно удоствериться что сторонним приложениям дан доступ 
          к использованию почты
          
          для этого заходим в google account
          переходим во вкладку БЕЗОПАСНОСТЬ
          переходим к окошку
          "ненадежные приложения у которых есть доступ к аккаунту"
          открываем доступ
          
          

Администрирование:

    - Создание обьявления
        
        ключевые моменты
        ! создавая обьяление нужно ОБЯЗАТЕЛЬНО загрузить
        1 картинку в главную картинку 
        и минимум 1 картинку в згрузчик нескольких картинок
        
    - удаление обьявления 
        
        переходим по пути 
        House/DeleteView
        
        копируем Id из url нужного дома
        пример:
        http://multihouse.world/House/HouseView/7
        тут Id = 7
        
        вставляем id 
        удаляем обьявление
        
        ! при удалении удаляются и данные
